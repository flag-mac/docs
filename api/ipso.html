IP查询归属地

# 简介

IP查询归属地，你可以放在网站对接使用。

# 接口说明

注意

如果您已开始使用本 API 服务，则默认视为遵守 本约定。

# 请求地址

地址 协议 方法 线路
https://api.gitcc.cn/ipso/ HTTPS Any 境内

# 使用示例

直接查询（本机IP）：https://api.gitcc.cn/ipso/

https://api.gitcc.cn/ipso/ (opens new window)

指定查询（指定IP）：https://api.gitcc.cn/ipso/?ip=8.8.8.8

https://api.gitcc.cn/ipso/?ip=8.8.8.8 (opens new window)
